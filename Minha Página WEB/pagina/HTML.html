Conhecendo o HTML

Explicação ultra, superficial

Em nossa primeita aula o professor, começou a explicar a história do HTML,falando a respeito da sua data de criação; o porque de utilizalo; o fim para qual o mesmo foi desenvolvido e quão importante são as suas funcionalidades. O HTML não é uma linguagem em sí e sim um tipo de arquivo, composto de TAGS que auxiliam na formatação de uma página WEB. Existem (n), tipos de TAGS que podem ser utilizadas no desenvolvimento de um documento do tipo HTML. Nesse mundo se o senhor(a) conhece bem o HTML, consegue desenvover páginas mais organizadas, configuradas da melhor forma possível e ainda agraveís para os olhos de quase todo o usuario.Porém o HTML não trabalha sózinho, existem outras tecnologias que auxiliam no desenvolvimento de uma página WEB, como por exemplo: CSS; JavaScript, JQuery, PHP, etc...
